# GamelWise 🛡️
Premium financial tracking and optimization for Israeli provident funds (Kupot Gamel).

## Features
- **PDF Parsing Simulation:** Visualize your data from quarterly reports.
- **Market Comparison:** Compare your fees and returns against market averages.
- **Savings Calculator:** See exactly how much you can save by lowering management fees.
- **Responsive Design:** Premium UI built with Tailwind CSS and Chart.js.

## How to use
1. Upload your HTML file to GitHub.
2. Enable **GitHub Pages** in the repository settings.
3. Access your personalized dashboard via the provided URL.

## Built with
- HTML5 / Javascript
- Tailwind CSS (CDN)
- Chart.js
